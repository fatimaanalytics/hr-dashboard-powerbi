# HR Analytics Dashboard (Power BI)

Project Overview : 
This project presents an interactive HR analytics dashboard built using Power BI to support management in monitoring and analyzing workforce data.
The goal was to create a centralized solution for tracking key HR metrics and enabling data-driven decision-making.

Business Problem :
The organization did not have a unified dashboard to track important HR metrics such as:
. Headcount
. Attrition rates
. Workforce distribution
As a result, reporting was manual and lacked clear insights for leadership.

My Role :
. Collaborated with stakeholders and managers to define key KPIs
. Gathered business requirements for dashboard design and insights
. Translated business needs into technical solutions

Data Extraction & Preparation :
. Wrote SQL queries to extract employee data from multiple tables
. Combined and structured data from different sources
. Ensured data accuracy and consistency

Data Cleaning & Transformation (Power BI) :
. Cleaned and validated data in Power BI
. Created calculated columns and date-related fields
. Built a structured data model for reporting

Dashboard Features :
. Headcount analysis
. Monthly and rolling attrition rate
. Employee distribution by department and gender
. Time-based trends and insights

Tools & Technologies :
. SQL
. Power BI
. DAX (Data Analysis Expressions)
. Data Modeling

Key Achievements :
. Developed a fully automated dashboard with monthly updates
. Eliminated manual reporting processes
. Delivered clear and interactive insights for management

Data Disclaimer :
The data used in this project has been anonymized and modified to protect confidentiality.
This project is based on a real business use case.

Future Improvements :

. Add more advanced HR metrics (e.g., retention analysis, tenure insights)
. Enhance data model scalability
. Improve dashboard interactivity
